Quick Mental Model
Core Philosophy

The Scout treats every matchup the way a real analyst would. Where efficiency models see numbers, it sees context: the injured point guard, the road-weary roster, the first-year coach about to crack.

Key Inputs
  • Coaching experience — tournament wins, Final Four appearances
  • Roster age — senior vs. freshman distribution
  • Injury / availability — beat reporters + official logs
  • Clutch performance — record in games decided by ≤5 pts
  • Momentum — last 10 games, conference tournament
  • Travel and rest — distance, days between games
Strengths
  • Captures intangibles pure efficiency models miss
  • Contextual synthesis of 6 factors simultaneously
  • No normalization required — model understands magnitude natively

Methodology
The Actual Build
The Process
Pull team data
Coaching record, roster age, injury log, recent results, travel schedule for each team.
Build matchup context objects
Pair Team A vs. Team B into a structured JSON block for each of the 32 first-round games.
Run structured analyst prompt
Feed context to Claude with analyst role. Returns scouting report + confidence-weighted pick in JSON.
Advance bracket round by round
Winners become context for the next round. Repeat through the championship.
Lock picks March 19
Final bracket exported to JSON. ESPN Tournament Challenge submitted as timestamp proof.
Data Sources
  • KenPom — efficiency ratings, 2026 season through conference tournaments
  • Sports Reference — coaching records and tournament history
  • Beat reporter injury coverage (ESPN, team beat writers)
  • Official injury designations where available
Key Assumptions
  • LLM contextual synthesis outperforms regression on the same inputs
  • Coaching experience is a consistent predictor across eras
  • Injury data 48 hours before lock is sufficiently current
Limitations
  • Hallucination risk on player stats — mitigated by injecting verified data
  • LLM bias toward brand-name programs — prompted against, imperfectly corrected
  • No live updates after March 19
