# 🏀 The Bracket Lab

**5 AI Models. 1 Tournament. Real-Time Tracking.**

An experiment pitting five fundamentally different AI prediction approaches against each other in the 2026 NCAA Tournament, with real-time accuracy tracking.

## The Models

| # | Model | Approach | Tagline |
|---|-------|----------|---------|
| 1 | **The Scout** | LLM Matchup Analysis | *"Film room intelligence at machine speed."* |
| 2 | **The Quant** | Monte Carlo Simulation | *"10,000 simulations. Zero feelings."* |
| 3 | **The Historian** | Archetype Matching | *"Every team has a twin. History already played this game."* |
| 4 | **The Chaos Agent** | Upset Detection | *"Your bracket is too safe. This one isn't."* |
| 5 | **The Agent** | Autonomous AI Research | *"No rules. No prompts. Just: build me a bracket."* |

All picks locked before the Round of 64 and verified via ESPN Tournament Challenge.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Content:** MDX
- **Hosting:** Vercel
- **Model Scripts:** Python + Claude API + Claude Code
- **Data:** KenPom, BartTorvik, ESPN

## Getting Started

```bash
git clone https://github.com/willclayton/the-bracket-lab.git
cd the-bracket-lab
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
