# Scout Prime — Data Source Manifest

## Primary Data Sources

### BartTorvik / T-Rank
- **What:** Advanced efficiency metrics for every D1 team
- **Fields:** AdjO, AdjD, AdjEM, Tempo, Barthag, WAB, Elite SOS, SOS
- **Location:** `optimizer_agent/research/barttorvik_YYYY.csv` (shared, read-only)
- **Temporal:** Per-season, available before tournament starts
- **Verified:** Yes — BartTorvik is a standard public source

### Tournament Bracket Structure
- **What:** R64 matchups (teams, seeds, regions) from actual-results.json
- **Location:** `data/archive/YYYY/results/actual-results.json`
- **Used for:** Extracting matchup structure ONLY (teams/seeds/regions)
- **NOT used for:** Game winners or scores (those are outcomes, not inputs)

### Historical Tournament Teams
- **What:** Past tournament teams with their efficiency profiles and tournament results
- **Location:** `data/research/historical-teams.json` or built from BartTorvik archives
- **Used for:** Archetype matching (finding statistical twins)
- **Temporal isolation:** For year Y, only use teams from years < Y

### Seed History
- **What:** Historical win rates by seed matchup type
- **Location:** `data/research/seed-history.json`
- **Used for:** Base rates for seed-based upset likelihood

### Upset Factors
- **What:** Pre-computed vulnerability scores by seed pairing
- **Location:** `data/research/upset-factors.json`
- **Used for:** Calibrating upset predictions

### Team Metadata
- **What:** Additional per-team context (coaching records, roster info, injuries, etc.)
- **Location:** `data/meta/teams.json`
- **Status:** Partially populated — full data added when bracket drops March 15

### Intangibles Intelligence
- **What:** Qualitative/circumstantial factors per team (social media activity, motivation, logistics, chemistry, health, local buzz, preparation, wildcards)
- **Location:** `data/research/intangibles_YYYY.json` (shared data source)
- **Generated by:** `scout_prime_agent/src/gather_intangibles.py`
- **Temporal:** Gathered 24-48 hours before picks lock (for live year). Reconstructed from Claude's training data for backtest years.
- **Categories:** social_media, motivation, logistics, chemistry, health, local_buzz, preparation, wildcards
- **Per team:** overall_vibe_score (1-10), intel items with category/signal/impact, summary, red_flags, tailwinds
- **Modes:** Live (`--year 2026`) or reconstructed (`--reconstructed` flag, auto-enabled for years < 2026)
- **Optional:** Pipeline works without this file — `enrich_teams.py` gracefully skips if not present

## Generated Research Data (Scout Prime)

| File | Generated By | Content |
|------|-------------|---------|
| `intangibles_YYYY.json` | `gather_intangibles.py` | Qualitative intel per team (shared, in `data/research/`) |
| `teams_enriched_YYYY.json` | `enrich_teams.py` | Merged team profiles (BartTorvik + metadata + intangibles) |
| `historical_archetypes_YYYY.json` | `build_archetypes.py` | Top 3 historical twins per team |
| `matchup_contexts/r64_YYYY.json` | `build_matchup_contexts.py` | R64 context packets |

## Data Integrity Rules

1. **Results are NEVER in prompts.** Actual game winners/scores are only used post-prediction for scoring.
2. **Temporal isolation enforced.** Archetype matching for year Y only uses data from years < Y.
3. **BartTorvik data is pre-tournament.** Reflects regular season + conference tournament, not tournament outcomes.
4. **All sources documented here.** If a data source isn't listed, it shouldn't be used.
