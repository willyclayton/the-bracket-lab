Option C: Icon + left border + "over" format
- Category icon on the left (checkmark / lightning / warning / swords) replaces the seed-matchup label.
- Left border color reinforces the category.
- "Team over Team" format with seed numbers inline — reads like a pick, not a matchup chart.
- Contested games use "vs" instead of "over" (since there's no clear consensus).
- Clean games: no icon, no border, just the agreement pill.
- More scannable than A (icon is faster to parse than text tag), more informative than B (icon + border vs border alone).
